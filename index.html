Projects

Agrosha

Agrosha is a machine learning-based web application designed to help farmers and agricultural analysts predict crop yield with greater accuracy. The app uses a two-stage model pipeline: the first model predicts key factors like Production, Rainfall, Fertilizer usage, and Pesticide levels based on inputs such as Crop, State, Season, Area, and Year. These predictions are then passed into a second model that estimates the final crop yield.

Agrosha is built using Python (Flask) for the backend and integrates trained ML models stored as `.pkl` files. The app streamlines complex agricultural data into a user-friendly interface, making it easier for users to make informed decisions. This project reflects my passion for solving real-world problems through technology and my growing experience in full-stack development and machine learning.

Technologies: Python, Flask, Scikit-learn, HTML/CSS, JS

Desi Traveller (WIP)

Desi Traveller is an innovative platform that combines modern AI technologies with interactive user experiences. At its core is an intelligent chatbot assistant that helps users seamlessly explore destinations, make travel bookings, and get personalized trip suggestions — all through natural conversation.

The website also features live updates on trending, ongoing, and upcoming events, ensuring that users never miss out on experiences worth attending. One of the standout features of Desi Traveller is its cutting-edge **hologram integration**, offering a futuristic preview of destinations and events, making trip planning more immersive than ever.

Features: AI Chatbot, Event Tracker, Hologram Previews

Tech: Node.js, Express, MongoDB, OpenAI API, WebXR
